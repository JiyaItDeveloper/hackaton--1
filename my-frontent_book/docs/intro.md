# All Modules Overview

Welcome to the **Technical Documentation Hub** - your comprehensive resource for ROS 2 and Docusaurus-based technical authoring. This hub contains all modules with their specifications and chapters.

## Module Specifications & Chapters

### 1. ROS 2 Module: The Robotic Nervous System
**Module Goal:** Understand ROS 2 as the nervous system of a humanoid robot, enabling communication between Python AI agents and robot controllers.

**Learning Objectives:**
- Understand the purpose and architecture of ROS 2 in Physical AI systems
- Implement ROS 2 communication patterns (Nodes, Topics, Services, Actions)
- Create Python AI agents that interface with ROS 2 controllers
- Model humanoid robots using URDF (Unified Robot Description Format)
- Design message flows between AI logic and robot controllers

**Chapters:**
- [Chapter 1: ROS 2 Fundamentals](./ros2-module/chapter-1/fundamentals) - Introduction to ROS 2 concepts and architecture
- [Chapter 2: ROS 2 Communication Model](./ros2-module/chapter-2/communication-model) - Nodes, Topics, Services, and Actions
- [Chapter 3: Python AI Agents with ROS 2](./ros2-module/chapter-3/ai-agents) - Integration of AI algorithms with ROS 2
- [Chapter 4: Humanoid Modeling with URDF](./ros2-module/chapter-4/urdf-modeling) - Robot modeling and visualization
- [Integration Project](./ros2-module/integration-project/integration) - Complete ROS 2 system implementation
